# 数据库课程设计 后端
# requirement
* python 3.8+
* PostgreSQL 14

## 安装库
在项目根目录下
```powershell
pip install -r requirements.txt
```
## 创建表
在项目根目录下
```powershell
psql.exe -U postgres -d tb -f "src\sql\1_create_table.sql" 
psql.exe -U postgres -d tb -f "src\sql\3_create_table.sql" 
```

# 启动server
在项目根目录下
```powershell
uvicorn src.main:app --reload 
```
